# FixIt 🛠️
> Stop filing forms. Just show FixIt.

AI-powered community issue reporting platform built for **Vibe2Ship Hackathon** by Coding Ninjas × Google.

## 🔗 Live Demo
[fixit-nine-sand.vercel.app](https://fixit-nine-sand.vercel.app)

## 🎯 Problem Statement
Community Hero - Hyperlocal Problem Solver

## ✨ Features
- **AI Image Analysis** — Gemini Vision auto-categorizes issue and detects severity
- **Interactive Map** — Live pins for all reported issues
- **Community Verification** — Upvote to verify issues
- **Autonomous Escalation Agent** — Auto-generates formal complaint letter to Municipal Corporation at 3+ verifications
- **Real-time Dashboard** — Total, escalated, and resolved counts
- **Image Validation** — Rejects non-civic photos
- **Resolution Tracking** — Mark issues as resolved

## 🛠️ Tech Stack
- React.js + Tailwind CSS
- Firebase Firestore
- Leaflet.js (Maps)
- Gemini 2.5 Flash (AI Vision + Text)
- Google AI Studio

## 🚀 Run Locally
```bash
git clone https://github.com/Pratyush0206/Fixit.git
cd Fixit
npm install
# Add your REACT_APP_GEMINI_API_KEY to .env
npm start
```
